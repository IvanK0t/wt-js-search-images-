### 3.3 Реалізація API модуля

**Файл: src/js/api.js**

```javascript
import axios from 'axios';

// Pexels API для пошуку зображень
const PEXELS_API_KEY = 'ваш_ключ_тут';
const BASE_URL = 'https://api.pexels.com/v1/search';

export async function fetchImages(query, page = 1) {
  try {
    const params = {
      query,
      page,
      per_page: 40,
    };

    const config = {
      headers: {
        'Authorization': PEXELS_API_KEY,
      },
    };

    const response = await axios.get(BASE_URL, { params, ...config });
    
    // Трансформація даних у внутрішній формат
    return {
      hits: response.data.photos.map(img => ({
        id: img.id,
        webformatURL: img.src.medium,
        largeImageURL: img.src.large,
        tags: img.photographer || 'Photo',
      })),
      totalHits: response.data.total_results || 0,
    };
  } catch (error) {
    console.error('Помилка API:', error.message);
    throw error;
  }
}
```

### 3.5 Логіка пошуку та пагінації

**Файл: src/main.js**

```javascript
// Змінні стану
let currentQuery = '';
let currentPage = 1;
let isLoading = false;

// Функція пошуку
async function searchImages(e) {
  e.preventDefault();
  
  const query = document.querySelector('input[name="query"]').value.trim();
  
  if (!query) return;
  
  currentQuery = query;
  currentPage = 1;
  gallery.innerHTML = '';
  
  try {
    const data = await fetchImages(query, currentPage);
    renderGallery(data.hits);
    
    // Показуємо кнопку Load More, якщо є сторінки
    if (data.totalHits > 40) {
      loadMoreBtn.hidden = false;
    }
  } catch (error) {
    console.error('Помилка пошуку:', error);
  }
}

// Функція завантаження ще
async function loadMore() {
  if (isLoading) return;
  
  currentPage++;
  isLoading = true;
  
  try {
    const data = await fetchImages(currentQuery, currentPage);
    renderGallery(data.hits);
  } finally {
    isLoading = false;
  }
}

// Рендеринг галереї
function renderGallery(images) {
  const html = images.map(img => `
    <a href="${img.largeImageURL}" class="gallery-item">
      <img src="${img.webformatURL}" alt="${img.tags}" />
    </a>
  `).join('');
  
  gallery.insertAdjacentHTML('beforeend', html);
}

// Обробники подій
document.querySelector('#search-form').addEventListener('submit', searchImages);
document.querySelector('#load-more').addEventListener('click', loadMore);
```

## 4. КОНТРОЛЬНІ ЗАПИТАННЯ ТА ВІДПОВІДІ

### 4.1 Протоколи HTTP та HTTPS

**Запитання:** Яка різниця між HTTP та HTTPS?

**Відповідь:** 
- **HTTP** (HyperText Transfer Protocol) - це протокол передачі даних без шифрування
- **HTTPS** (HTTP Secure) - це HTTP з додатковим шифруванням TLS/SSL
- HTTPS забезпечує безпеку передачі даних, захищає від перехвату
- Сучасні API сервіси (Pexels, Unsplash, Pixabay) використовують HTTPS

### 4.2 Формат запиту та HTTP-методи

**Запитання:** Назвіть основні HTTP-методи та їх призначення?

**Відповідь:**
- **GET** - отримання даних (використаний у проекті для пошуку)
- **POST** - створення нових ресурсів
- **PUT** - оновлення існуючих ресурсів
- **DELETE** - видалення ресурсів
- **PATCH** - часткове оновлення

**Приклад запиту в проекті:**
```javascript
axios.get('https://api.pexels.com/v1/search', {
  params: { query: 'cat', page: 1, per_page: 40 },
  headers: { Authorization: 'API_KEY' }
});
```

### 4.3 AJAX та Fetch API

**Запитання:** Як AJAX та Fetch API відрізняються від традиційного завантаження сторінок?

**Відповідь:**
- **Традиційно** - при кожному запиті перезавантажується вся сторінка
- **AJAX** (Asynchronous JavaScript And XML) - дає можливість асинхронно завантажувати дані без перезавантаження
- **Fetch API** - сучасна заміна AJAX, заснована на Promise
- У проекті використаний **axios** (обгортка над Fetch API) для зручності

**Порівняння:**
```javascript
// Fetch API
fetch(url).then(response => response.json()).then(data => {...});

// Axios (у проекті)
axios.get(url).then(response => {...});
```

### 4.4 REST API та параметри запиту

**Запитання:** Що таке REST API та як передаються параметри?

**Відповідь:**
- **REST** (Representational State Transfer) - архітектурний стиль для побудови веб-сервісів
- **Параметри рядка запиту** передаються через знак `?`:
  ```
  https://api.pexels.com/v1/search?query=cat&page=1&per_page=40
  ```
- **Заголовки** передають метаінформацію (у т.ч. API ключ):
  ```
  Authorization: 'API_KEY'
  ```

### 4.5 Пагінація

**Запитання:** Як реалізована пагінація в проекті?

**Відповідь:**
- Кожен API запит має параметри `page` та `per_page`
- При першому пошуку `currentPage = 1`
- При кліку на "Завантажити ще" `currentPage++`
- Нові результати додаються до галереї без перезавантаження сторінки
- Кнопка приховується, коли досягнуто кінця результатів

```javascript
// Пагінація
currentPage++;
const data = await fetchImages(query, currentPage);
gallery.insertAdjacentHTML('beforeend', renderGallery(data.hits));
```
